# reference/docs/extracted/

Verbatim vendor extracts live here, one subdirectory per source
(`extracted/<source-slug>/`). See RULE-017 (`rules/distilled-doc-provenance.md`):
unmodified vendor text lifted from a larger source goes here and is
immutable once added (RULE-016); our digests of it live in `context/`
and cite the extract path.

Empty (aside from this marker) until the first extract lands.
